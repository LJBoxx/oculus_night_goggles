# Oculus cv1 NVG
This repo contains code to interface with an oculus cv1 kit in order to repurpose it into night vision goggles.
It is made up of 2 main binaries, one that enable extended mode in the headset and the other that displays the frames captured by the sensor.

## Hardware
In order to run this, you will need to have an oculus cv1 heaset and two Oculus Constellation sensor (basically infared webcams) if you have regular camera you could simply use it with the hmd binary.
